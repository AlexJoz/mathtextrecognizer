Reconocedor de texto matemático
===============================
Desarrollado como proyecto de fin de carrera por Luis Román Gutiérrez,
a partir de un trabajo de PID desarrollado por Luis Román Gutiérrez,
Irene Román Rubio, Javier Valdés Lopez, Fidel Ramos Sañudo.

Requisitos
==========
Para ejecutar este programa se necesita tener instalado el runtime de Mono 1.9
o superior junto con GTK# 2.10 o superior.

El ejecutable de Mono "mono", debe estar en el Path de su sistema.

Nota:
     Este proyecto utiliza iconos de los proyectos Tango y Pasodoble.
